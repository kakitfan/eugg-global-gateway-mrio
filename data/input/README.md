# Local Input Data

Put model input files in this directory before running the pipeline.

This directory is intentionally ignored by Git. The repository does not
redistribute GLORIA matrices, local investment workbooks, or generated result
workbooks.

Required files:

- `FD2023.mat`
- `UT2023.mat`
- `V2023.mat`
- `va_f.mat`
- `CO2_2023.mat`
- `C_matrix.mat`
- `pop_weights.mat`
- `country_population_weights_wb_2023.csv`
- `Data.xlsx`
- `World Bank Country Class_2025_10_07.xlsx`
- `Investment matrix_GLORIA.xlsx`
